Sobre Mim

Olá! Sou estudante do curso técnico de Informática e estou aprendendo a programar, dando os meus primeiros passos no desenvolvimento de software. Como sou iniciante e ainda não sou muito bom, o meu foco principal agora é entender como criar páginas simples usando HTML para organizar os dados e CSS para deixar o visual bonito e arrumado.

Busco transformar linhas de código em interfaces organizadas, agradáveis e funcionais, evoluindo a cada aula prática e projeto desenvolvido.

Objetivos Profissionais

Meu principal objetivo no momento é aprender bem a base e consolidar uma base sólida em lógica de programação. No futuro, pretendo ingressar no mercado de trabalho como Engenheiro, colaborando em grandes projetos e aplicando tecnologia para resolver problemas reais do cotidiano.

O que estou Aprendendo

  • Versionamento de Código: Uso inicial de Git e plataforma GitHub para gerenciar e hospedar códigos com segurança.
  • Lógica e Resolução de Problemas: Construção de algoritmos e desenvolvimento do raciocínio analítico computacional.
